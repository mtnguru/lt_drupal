INTRODUCTION

--------------

  Most of the developers I know simply hate Light IDE.However, Drupal Admin
interface is OUR IDE and it is Light! Bartik, Seven and now Claro all goes to
Light. Why not changing to one of the most famous dark theme of all time:
Darcula? That's the purpose of this Admin Theme: a Claro theme with Darcula
colors.

REQUIREMENTS

--------------

  This module requires no modules outside of Drupal core.

INSTALLATION

---------------

*  Install as you would normally install a contributed Drupal theme. Visit
https://www.drupal.org/docs/user_guide/en/extend-theme-install.html for further
information.


CONFIGURATION

--------------


  This theme has no the special configuration. We need just to install and
to set as default on appearence menu in your CMS. After that this theme will
works.
